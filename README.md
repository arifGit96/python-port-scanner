# Python Port Scanner

A multithreaded TCP port scanner built with Python.

## Features

* TCP Port Scanning
* Hostname Resolution
* Service Detection
* Banner Grabbing
* Multithreaded Scanning
* TXT Report Export

## Technologies

* Python
* Socket Programming
* ThreadPoolExecutor

## Usage

bash
python Port-scanner.py


## Example Output

text
[+] Port 21 OPEN (FTP)
Banner: MikroTik FTP Server

[+] Port 1723 OPEN (PPTP)
```
